## 二、运算符优先级

### 2.1、原码、反码、补码
- 正数：二进制原码、反码、补码相同
- 负数：
  - 原码：符合位为1
  - 反码：符号位不变，其余位取反
  - 补码：反码+1
### 2.2、位运算符（使用补码进行运算）
- ~（位非）：~x = -(x+1)
  - 正数：所以位全部取反，得到补码 -> 补码转反码 -> 反码转原码
  - 负数：原码转反码 -> 反码转补码 -> 所以位全部取反
- &（位与）：相同为1，不同为0
- |（位或）：有1为1，没有为0
- ^（位异或）：相同为0，不同为1
- \>\>（右移）：
  - 正数：直接进行位移，高位补0
  - 负数：补码进行位移，高位补1 -> 转反码 -> 转原码
- <<（左移）：
  - 正数：直接进行位移，低位补0
  - 负数：补码进行位移，低位补0 -> 转反码 -> 转原码
### 2.3、is和==
- ==比较的是两个变量的值
- is比较的是两个变量的内存地址
- 当比较的两个不可变类型（str）的对象，==和is是等价的；当比较两个可不类型（list，dict，tuple）的对象，==和is是有区别的
## 三、变量和赋值
- 变量在使用前，要先对其赋值
- 变量名可以包括字母、数字、下划线，但是不能以数字开头
- python变量名对大小写敏感

## 五、位运算操作
### 5.1、<<、>>快速计算2的倍数问题
- n << 1 -> n*2
- n >> 1 -> n/2，负奇数的运算不可用
- n << m -> n * 2**m
- n >> m -> n / 2**m
